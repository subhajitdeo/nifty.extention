# Nifty OI Levels Extension

## Features
- Automatically draws OI levels on NSE chart (charting.nseindia.com)
- Updates every 10 seconds
- Manual backup: copy console code if extension fails

## Installation
1. Download all files into a folder
2. Go to chrome://extensions/
3. Enable Developer mode
4. Click "Load unpacked" and select the folder

## Usage
- Open https://charting.nseindia.com/
- Lines will appear automatically
- Click extension icon for manual controls

## Backup
If extension stops working, click "Copy Console Code" and paste in DevTools console.
